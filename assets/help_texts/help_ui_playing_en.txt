# CONTEXT:nowplaying_mode
OK: Open menu
OK long: Open Tools menu
CH +: Add Favorites / SongLog if radio
CH -: Remove current track from Queue
↑ ↓: Increase/Decrease Volume
← →: Previous/Next Track
← → long: Rewind/forward 10s
↵: Playlist screen
↵ long: Library navigation screen
power: Shutdown
power long: Reboot
Info: Show/Close Help
